En esta carpeta se encuentra el script de R con el cual se realiza el calculo del IPP.

El script principal es un R notebook que se llama IPP. El archivo funciones.R contiene las funciones que se utilizan para realizar el calculo. 

Hay dos carpetas una llamada 'índices' en la cual se guardan los resultados en archivos de Excel de los resultados del calculo mensual del IPP. La otra carpeta se llama 'pruebas' y contiene algunos scripts que se usaron como prueba de calculo previo a tener el archivo principal.

El archivo de Excel 'catalogoIPP_test' tiene el catalogo del IPP con la ultima versión hasta el 27/9/2024, asi mismo el archivo 'precios_test' contiene la boleta de los precios actualizados hasta la fecha.

Por el momento el archivo en la carpeta 'indices' llamado indice_DICIEMBRE, contiene los índices de valor 100 ya que es el mes base para realizar el calculo.
